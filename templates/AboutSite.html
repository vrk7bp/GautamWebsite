About The Site

So what's under the hood...




"Success is a journey, not a destination. The doing is often more important than the outcome."

- Arthur Ashe


If you haven't seen it already, the code for this site can be found here. This site is built using a Python Framework called Flask. For anyone wanting to build their own site from scratch, I highly recommend using Flask. It is extremely easy to use and very intuitive. From a server standpoint, this site is running on Google App Engine. Although the App Engine documentation makes it sound like it is very easy to run a Flask webserver on their machines, I did not find this to be the case. I spent many hours trying to figure out how to run Flask on Google's platform, and even considered jumping to Amazon Web Services a couple of times. One of the biggest problems is that unless the 3rd party application you want to use is on this list, you have to physically download these files and put them in the right hierarchy among your other folders and files related to the project. This got especially annoying with Flask, because Flask is built on other un-supported 3rd party libraries such as werkzeug and jinja2. Eventually I was able to get the web server working, then it was a matter of figuring out the HTML and CSS. For those of you who aren't interested in learning how to use Flask with Google App Engine, or don't want to deal with a webserver at all, I recommend looking into GitHub Pages. GitHub Pages is perfect for those people who are more worried about the HTML, CSS, and JavaScript because it abstracts away the Web Server.

Based upon the troubles I had, and the amount of searching I had to do to find the right answers, I figured it would be worthwhile to create a step-by-step tutorial on how to build a Flask webserver on App Engine. Although there are ready-to-go GitHub repositories that you can simply fork and use, I did not find this as satisfying as figuring it out myself. There were a couple of solutions I happened on that turned out to be outdated, and it became hard trying to figure out what was old information from new. My goal will be to have step by step instructions, as well as explanations as to what you are doing, for building a Flask web server on Google App Engine. I will also throw in some explanations on how to get your own personal URL for the site (I used NameCheap), and how to link the code you deploy to App Engine to GitHub so you can push to both at the same time. This will all be under the pre-tense of building your own personal website. Stay tuned...

So why did I build this site? The more and more I have gotten into programming, the quicker I have realized that I learn by doing. There is only so much I can get out of reading a tutorial or typing some lines on Codecademy (not that I dislike Codecademy, I love the site and visit it often). At some point you just have to really dive in. This site was my version of diving into web development. I had learned about building a basic web server and the power of Flask before, but I had absolutely no knowledge of building websites and figured this would be the perfect opportunity to learn. Throw in the fact that I learned a bit about Google App Engine, and I would say that it was an overall great experience that was well worth while.

The other reason I built this site is because every legitimate, boss programmer has his or her own website. And I was ready to become a boss programmer.

If you are working on building your own website and have any questions, feel free to shoot me an e-mail. I can't guarantee that I will have an answer, but I would love to try and help.
